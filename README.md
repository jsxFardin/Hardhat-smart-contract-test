## Test Smart Contract With Hardhat

1. Node install

```
npm i
```

2. To run Hardhat

```
npx hardhat
```

3. Compile smart contract

```
npx hardhat compile
```

4. Test smart contract 

```
npx hardhat test
```

4. Deploy smart contract 

```
npx hardhat run scripts/deploy.js --network <network-name>
```
